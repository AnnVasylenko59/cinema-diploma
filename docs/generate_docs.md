# Інструкція з генерації документації проєкту "Cinema-Diploma"

Цей проєкт використовує **JSDoc** для автоматичної генерації технічної документації з коментарів у коді.

## Попередні вимоги
- Встановлений Node.js.
- Встановлені залежності (`npm install`).

## Як згенерувати документацію
Виконайте команду в кореневій директорії проєкту:
`npm run docs`

## Перегляд результату
Після виконання команди документація буде доступна в папці:
`./docs/gen/index.html`

Відкрийте цей файл у будь-якому сучасному браузері.
