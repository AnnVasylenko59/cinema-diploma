Інформаційна вебсистема онлайн-кіноафіші

Information Web System of an Online Cinema Schedule

Сучасна вебплатформа для моніторингу кіносеансів, взаємодії з інтерактивною картою та автоматизації процесу бронювання квитків.

Актуальність теми

Автоматизація взаємодії глядача з кінотеатрами є критично важливою для покращення користувацького досвіду. Даний проєкт вирішує проблему швидкого доступу до розкладу та зручного вибору місць через інтерактивний інтерфейс.

Мета та завдання

Мета: Розробка масштабованої вебсистеми з інтерактивною мапою сеансів та персоналізованими рекомендаціями.

  • Проєктування реляційної бази даних з використанням Prisma ORM.
  • Розробка серверного API на базі Node.js та Express.
  • Створення реактивного інтерфейсу користувача за допомогою React.
  • Інтеграція картографічних сервісів для візуалізації кінотеатрів.

Методологія дослідження

В основі проєкту лежить архітектура Monorepo, що дозволяє забезпечити цілісність коду фронтенду та бекенду. Для розробки обрано стек Fullstack JavaScript, що забезпечує високу швидкість обміну даними та зручність масштабування.

Очікувані результати

Створення робочого прототипу системи, що забезпечує в реальному часі моніторинг сеансів, стабільне бронювання місць та коректне відображення на мобільних пристроях.
